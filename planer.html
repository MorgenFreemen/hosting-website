Dev-pakken

Dette webhotel dækker alle behov for nybegyndere og mindre hjemmesider. Det er oftest brugt til at udvikle på, men kan sagtens bruge til at køre en hel hjemmeside!

Basis-pakken

Perfekt til større hjemmesider, der kræver mere plads. Hurtigt og med fantastisk oppetid.

Ultimativ-pakken

Ultra hurtigt webhotel der er perfekt til de største hjemmesider og dem, der er mere driftkritiske.

Opgrader/nedgrader

Vi binder ikke vores kunder til den samme pakke i valgte periode. Du kan når som helst opgradere/nedgradere dit webhotel uden binding.


Delt dedikeret server

Din hjemmeside kommer til at elske at ligge på en delt dedikeret server! Høj performance og fantastisk oppetid.


Vælg selv PHP 5.4 - 7.2

Bestem selv hvilken version af PHP du vil bruge. Du kan også selv skrive din egen php.ini fil, hvis du tør.


1-click-installer

Hos os kan du nemt installere Wordpress, Joomla, Drupal, Prestashop, Magento og meget, meget mere! Vi tilbyder over 275 web-programmer.


Gratis SSL

Vi beskytter alle dine domæner med gratis SSL udstedt af Comodo. Dette gør vi hver nat helt automatisk, så du slipper for besværet!


Python & Ruby

Hos os kan du bruge dine Python og Ruby applikationer, samt installere moduler ganske nemt inde fra cPanel kontrolpanelet.


HTML Builder

Kom hurtigt i gang med en ny hjemmeside med Site Publisher HTML BUILDER, den finder du inde i cPanel kontrolpanelet.


Mobilvenligt

Med cPanels' nye tema Paper Lantern kan du nu nyde hele kontrolpanelet fra din computer, tablet og mobil helt uden problemer!
